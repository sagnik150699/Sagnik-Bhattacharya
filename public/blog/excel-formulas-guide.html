Excel Formulas: The Complete 2026 Reference

The complete Excel formulas reference — lookups, dynamic arrays, LAMBDA, Power Query, pivot tables, and advanced techniques with real examples.

This hub covers every major Excel formula topic with step-by-step tutorials, real examples, and honest trade-offs. It spans classic lookup functions like VLOOKUP and INDEX MATCH, the modern dynamic array functions (GROUPBY, PIVOTBY, MAP, SCAN, REDUCE), LAMBDA for reusable logic, and the productivity skills around them — pivot tables, Power Query, Power Pivot, dashboards, and error debugging.

Who This Guide Is For

This reference is designed for three audiences. If you are an absolute beginner who has only used SUM, COUNT, and IF, start with the Lookup Functions section — VLOOKUP is the gateway to everything else, and the VLOOKUP vs XLOOKUP comparison will immediately level you up. If you are an intermediate user comfortable with VLOOKUP and basic pivot tables, jump straight to the Dynamic Array Functions and Advanced Formulas sections — GROUPBY, PIVOTBY, and LAMBDA are the skills that separate power users from everyone else. If you are an advanced user or trainer looking for a reference to share with your team, use the Debugging and Productivity sections — these cover the operational skills (error tracing, structured references, keyboard shortcuts) that make the difference between a spreadsheet that works today and one that keeps working six months from now.

How to Navigate This Reference

The 23 tutorials below are organised into six sections that follow a natural learning path. The Lookup Functions section starts with the comparison every Excel learner eventually searches for — VLOOKUP vs XLOOKUP — and builds toward multi-criteria lookups with INDEX MATCH. The Dynamic Array Functions section covers the newest Excel features that replace many pivot table workflows with single formulas. If you are on Microsoft 365, these are the functions that will most dramatically change how you build spreadsheets. The Advanced and Reusable Formulas section bridges traditional Excel and modern approaches — LET and LAMBDA represent the most significant shift in how Excel formulas work since named ranges, and the Python integration extends what is possible inside a cell. The Pivot Tables and Data Analysis section covers the classic analysis stack. The Debugging section addresses the errors that waste the most time in real workbooks. The Productivity section covers the foundational habits — Tables, structured references, keyboard shortcuts — that make every other skill faster.

What You Absolutely Must Learn First

If you only have an hour, read these three tutorials in order: VLOOKUP vs XLOOKUP (because lookup functions appear in virtually every business spreadsheet), Excel Tables Best Practices (because Tables make every formula cleaner and more reliable), and LET and LAMBDA (because these two functions fundamentally change what is possible in a single cell). Everything else builds on these three skills. If you have a weekend, add GROUPBY and Pivot Tables — between them, they cover every summarisation scenario you are likely to encounter.

Key takeaways

  • XLOOKUP replaces VLOOKUP in most cases but INDEX MATCH still wins on multi-criteria and non-standard layouts.
  • Dynamic array functions (GROUPBY, PIVOTBY, MAP, SCAN, REDUCE) let you rebuild pivot-style reports as formulas that auto-update.
  • LAMBDA turns repeated formula logic into named, reusable functions — the closest Excel gets to programming.
  • Most #SPILL! errors come from obstructed ranges, not formula logic — clearing the spill range fixes them faster than rewriting the formula.
  • Excel Tables with structured references are more maintainable than A1:Z100 ranges in every practical workbook.
  • AI tools (Claude, Gemma 4, ChatGPT) can generate and debug formulas, but understanding what the formula does remains essential for production workbooks.
